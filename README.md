### 签名算法

- RSA：Rivest–Shamir–Adleman
- DSA：数字签名算法
- ECDSA：椭圆曲线数字签名算法，Elliptic Curve Digital Signature Algorithm
- EdDSA：爱德华兹曲线数字签名算法
- Ed25519：EdDSA 的一种实现方案，使用 SHA-512 / 256 + Curve25519
